# CyberBoard

A modern, cross-platform infinite digital whiteboard application built with .NET 9, Avalonia UI, and SkiaSharp.

## Features

- **Infinite Canvas** - Unlimited zoom, pan, and rotate
- **Drawing Tools** - Pen, pencil, marker, highlighter, eraser, shapes
- **Layer System** - Multiple layers with blend modes
- **Themes** - Light and dark mode
- **File Formats** - Native `.cboard` format, SVG export
- **Cross-Platform** - Windows and Linux native support

## Build

```bash
dotnet restore
dotnet build
dotnet run --project CyberBoard
```

## Requirements

- .NET 9 SDK
- Windows 10/11 or Linux (Wayland/X11)
